# NotesWay Frontend

## Overview
This is the **frontend application** for **NotesWay**, built using **React**.  
It provides the user interface for students, professors, and institutions to access, upload, and manage academic notes.

The frontend communicates with a **Python-based backend API** and is designed for scalability and production deployment.

---

## Tech Stack
- React
- JavaScript (ES6+)
- CSS Modules
- Axios
- React Router

---
